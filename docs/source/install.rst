Installing razorback
====================

Razorback works on all major platforms (linux, OSX, Windows).
It runs with Python>=3.12.

The stable version can be installed with:

.. code-block:: bash

  pip install razorback


Or it can be added to your project with:

.. code-block:: bash

  # uv project
  uv add razorback
  # pixi project
  pixi add --pypi razorback


The project repository is hosted on GitHub_.
You are welcome to contribute_.

.. _Github: https://github.com/BRGM/razorback
.. _contribute: https://github.com/BRGM/razorback/blob/main/CONTRIBUTING.md
